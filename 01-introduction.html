Goals

  • Master the fundamental concepts of the process of designing, implementing, and deploying a web application.
  • Proficient in HTML, CSS, Javascript.
  • Understand and build a website based on ReactJS.
  • Compare SSR and CSR, evaluate ReactJS state management & API integration.
  • Be able to deploy and manage web applications on the Internet.
  • Present, critique, and improve team projects based on feedback.

Assessment and Grading

  • Weekly homework: 10%
  • Group project: 50%
  • Midterm lab exam: 20%
  • Final lab exam: 20%

Tools

  • node, npm
  • git/github
  • Visual Studio Code
  • Postgres or any SQL database: MySQL/MariaDB, SQL Server (Not recommended)
  • TablePlus
  • Docker

Assignment Conventions

Must include these folders
  • Source: source code of submission.
  • Database: scripts of creating database structure and sample data.
  • Document:
    • Self-evaluate document
    • Functional description document
    • Instruction of database, environment setup
  • Demo (optional): video describe implemented function.
  • Ref (optional): reference used while doing assessment.
