JavaScript

JavaScript is a versatile, high-level, and interpreted programming language designed for the web. It allows us to create dynamic, interactive content on web pages, such as animations, form validations, and interactive menus. JavaScript works seamlessly with HTML and CSS, enabling us to build rich user interfaces and enhance the user experience.


						// This is a comment
						let message = "Hello, JavaScript!"; // Declares a variable
						console.log(message); // Outputs to the console

JavaScript

JavaScript is a client-side programming language that runs directly in the browser. While it is traditionally associated with the client side, modern technologies like Node.js have also introduced JavaScript as a server-side language. Its ability to manipulate the Document Object Model (DOM) makes it indispensable for modern web development.

  • Dynamic content: It can update HTML and CSS without reloading the page.
  • Interactivity: It supports event handling, animations, and user interaction.

DOM

The DOM (Document Object Model) is a programming interface that represents the structure of an HTML or XML document as a tree-like hierarchy. Each element, attribute, and piece of text in the document is represented as a node, enabling JavaScript to interact with and manipulate the document programmatically.

HTML provides the static structure of a web page, but the DOM acts as its live representation. When a browser loads an HTML document, it parses the content into the DOM, which allows JavaScript to dynamically update the page’s content and structure.

DOM

Accessing elements in the DOM is a fundamental operation that allows us to manipulate web page content dynamically. Several methods are available for selecting elements.


						const header = document.getElementById('main-header');
						const items = document.getElementsByClassName('list-item');
						const paragraphs = document.getElementsByTagName('p');
						const firstItem = document.querySelector('.list-item');
						const allItems = document.querySelectorAll('.list-item');

DOM

The DOM’s tree structure allows us to navigate between nodes. Here are some common traversal techniques:


						const childElement = document.querySelector('.child');
						const parentElement = childElement.parentNode;
					

						const parentElement = document.getElementById('parent');
						const children = parentElement.children;
					

						const current = document.querySelector('.current');
						const next = current.nextElementSibling;

DOM

We can use the textContent property to update or retrieve the textual content of an element.


						const heading = document.querySelector('h1');
						heading.textContent = 'Welcome to JavaScript!';

DOM

The setAttribute() and getAttribute() methods let you manipulate an element’s attributes.


						const link = document.querySelector('a');
						link.setAttribute('href', 'https://example.com');
						console.log(link.getAttribute('href'));

DOM

The style property allows you to modify inline CSS styles directly on an element.


						const button = document.querySelector('button');
						button.style.backgroundColor = 'blue';
						button.style.color = 'white';

DOM

Use the classList property to manipulate an element’s classes.


						const box = document.querySelector('.box');
						box.classList.add('highlight');
						box.classList.remove('shadow');
						box.classList.toggle('active');

The classList.contains() method checks if an element has a specific class.
